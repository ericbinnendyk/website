Golly scripts

All of these scripts should be run in Golly, a cellular automata simulator program.

  • glider_destruction: finds minimal glider destruction of a pattern
  • methuselah-searcher: naive script to find methuselahs in a given rule
  • polynomial-gun-generator: Can build patterns that emit gliders at generations that follow almost any polynomial sequence
  • similar-rule-finder: finds outer-totalistic rules that are the most similar to a given rule and outputs to a text file
