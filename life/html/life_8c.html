Parses a file containing a Life pattern in Life 1.05 format and stores it in a matrix.

Parameters
patfilea pointer to the file to read
patt_gridthe matrix to store the pattern in
widththe width of the matrix
heightthe height of the matrix
offsetthe offset of the pattern from the center of the grid
modean integer determining the behavior at the edges of the Life universe: hedge, torus, or Klein bottle.
Returns
1 if the file is badly formatted, 2 if the pattern is in a rule other than Life, otherwise 0
Remarks
The pattern read in from the file is processed in "virtual coordinates," where 0, 0 is at the center of the grid, as opposed to the coordinates used elsewhere that have 0, 0 in the upper left corner. This function is to be called by read_file, after the first line has already been read.
